About

Dasu.print
ALPHA
Version 0.3.0-alpha  ·  Build 2025  ·  Dasu (出す)

A free, open-source browser-based sheet layout tool — compose drawings from Bonsai BIM, SVG views, and images into print-ready sheets without leaving the browser.


Developed independently for the benefit of the OSArch and Bonsai communities. Not an official product of either community.


Built on Fabric.js (MIT) and jsPDF (MIT), works with IfcOpenShell.

GitHub Bonsai BIM IfcOpenShell OSArch

Setup
The bridge lets Bonsai BIM send drawings directly to Dasu without any manual file steps.
# Step 1 — run the bridge server
python dasu_bridge.py
# Step 2 — run the panel script in Blender Text Editor
dasu_panel.py → Run Script
# Step 3 — click Connect above, then in Blender:
N-panel → Dasu tab → Send to Dasu ↗
